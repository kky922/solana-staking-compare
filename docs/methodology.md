# Methodology

The comparison groups staking options by native staking, liquid staking, lending, restaking,
vaults, and liquidity pools. Static APY ranges are a dated reference snapshot. When a supported
public API responds, the live value is displayed separately and used for sorting.

Risk labels are qualitative:

- `LOW`: native staking or mature, highly liquid protocols
- `MEDIUM`: additional smart-contract or liquidity risk
- `HIGH`: leverage, concentrated liquidity, or newer protocols
- `VERY_HIGH`: complex or unstable yield sources

Users should verify official protocol documentation, current audits, liquidity, validator
performance, and local tax rules before making a decision.
